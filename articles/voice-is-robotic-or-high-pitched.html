If your voice or your guest's voice sounds robotic or high-pitched when recording on Alitu, this is usually because a microphone has a different sample rate than the computer's operating system. You can easily fix this by making sure the operating system is set to use the same sample rate as the mic.

  • On MacOS, launch the Audio and Midi Setup app and make sure the operating system sample rate is set to use 44.1KHz

  • On Windows, this is typically done by right-clicking the speaker icon in the system tray and selecting ‘Playback devices’ (or ‘Recording devices’), double-clicking the DVS devices, and changing the sample rate in the ‘Advanced’ tab for each device. (Make sure you are on 44.1KHz)

Please contact support if you have any issues switching this over.
