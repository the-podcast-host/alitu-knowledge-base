We are often asked what the 'best' audio format is to upload into Alitu.

For best quality, a .wav would be the standard starting point, if possible.

Wav files are higher quality, so always upload a .wav if you can. Starting with the best quality audio file will ensure that your final file will be the highest possible quailty.

MP3s are compressed files which means their file size is reduced by removing some of the detail. They still sound great, so it's fine to use them if that's what you have available.

At the moment we allow you to upload the following formats: .

m4a, .mp3, .flac, .mp4, .wav, .ogg, .aiff, .aif, .acc, .aac, .pcm, .wma, .alac, .mp4, .avi, .mov, .flv, .wmv, .mkv, .webm, .mka

Please reach out using the chat box at the bottom of this page if there's an audio format you want to use that you don't see above.

Also, let us know if you have any other questions about using Alitu. We're happy to help!
