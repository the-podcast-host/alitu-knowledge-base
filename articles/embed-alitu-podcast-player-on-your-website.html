How to embed an Alitu Podcast Player on your Website

Once your episode has been published, you can embed an episode-specific audio player on any website you own that accepts HTML code. The process for embedding the player will differ depending on what type of website you have, for example, WordPress versus Squarespace, but in each instance, you will copy your HTML embed code from your Alitu dashboard and paste that code into an HTML area on a blog post or page on your website.

1. After publishing your episode, click 'Details' to return to the Publish Page:

2. Click the ‘Copy embed code’ button:

3. Then copy the HTML embed code from the pop-up that appears:

5. Paste this code into your own website to embed the audio player.
